# Key Conflicts

Tracks macOS system shortcuts that our configuration overrides. Entries are added **only after** a conflict is actually introduced by a config change.

## Format

Each entry records:
- The shortcut being overridden
- What macOS natively uses it for
- What we reassigned it to
- Which tool introduces the conflict (Karabiner / Aerospace / Ghostty)
- Whether the native behavior was disabled or relocated
